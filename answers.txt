6. Find the shortest dinosaur, and rename it Shortie.
  >>> SELECT MIN(length) FROM dinos; --> This returns 0.08
  >>> SELECT name FROM dinos WHERE length = 0.08;
  name
-------------
Liaoxiornis
(1 row)
  >>> UPDATE dinos SET name = 'Shortie' WHERE length = 0.08;
  >>> SELECT name FROM dinos WHERE length = 0.08;
  name
---------
 Shortie
(1 row)
  >>> After running the update query, this is what the select query returns, which is exactly what we expected.
